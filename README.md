# trash-hackaton

для запуска клиента для разработки и тестов (нужен NodeJs):

<ul>
    <li>cd client</li>
    <li>npm i</li>
    <li>npm run start</li>
</ul>

# TelegramApp

для запуска бота нужен python3, а так же сторонние библиотеки json, numpy, telebot.

```bash
python install pip
pip install numpy
pip install json
pip install pytelegrambotapi
python main.py
```
