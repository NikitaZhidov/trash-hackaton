# trash-hackaton

для запуска клиента для разработки и тестов (нужен NodeJs):

<ul>
    <li>cd client</li>
    <li>npm i</li>
    <li>npm run start</li>
</ul>

# TelegramApp

для запуска бота нужен python3, а так же сторонние библиотеки json, numpy, telebot.

<ul>
    <li>'''$ python install pip</li>
    <li>'''$ pip numpy</li>
    <li>'''$ pip json</li>
    <li>'''$ pip telebot</li>
    <li>'''$ python main.py</li>
</ul>
